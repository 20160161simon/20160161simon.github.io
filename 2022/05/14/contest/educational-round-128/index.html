很简单的一题，如果区间重叠取大的 $l$ ，否则两个 $l$ 相加。

B

与A一样简单，若有一个机器人同时是最上面的也是最左边的就能成功，否则不行。

C

口胡了一个结论过了：

由于当你删到一定数量后 1 的数量会超过 0，那么我们没必要考虑 1 的数量超过 0 的情况，即只考虑 “删掉的 1” + “删掉的0” <= “0的数量即可”，于是删掉的数量猜测一定为 “0的数量”（直觉上是挺对的，但不会证明）O(n) 扫一遍就行。

代码保留了看错题目留下的痕迹。

由于 $n<=3000$ 显然可以直接枚举所有区间，然后尽量让前面 $a[i]=0$ 的点填得数大即可。

F

时隔多年再见二分图，debug 了好久。

题意等同于原图是否为二分图，如果不是能否通过删一条边让它是。

首先 dfs 判断是否为二分图，同时生成一个黑白染色树林，先当它只有一棵树。

如果这除了这棵 dfs 树上的边以外，我们把边分为好边和坏边，如果该边左右端点相同颜色是好边，否则是坏边。

显然如果只有一条坏边我们无需更改直接输出即可，否则按以下方法解决：

由于不止一条坏边我们就不能删 dfs 树以外的边了，因为这不会产生任何贡献。

为了删边后保持性质且具有贡献，我们删掉 dfs 树上的一边后，该边下面的子树应当全部翻转。于是我们有了子树 A 和子树 B 两颗树（假设子树 B 被翻转了）那么若所有坏边的端点都分别在 A 和 B 之间且 A 和 B 之间没有好边就可以了，否则不行。

题目转化为求所有坏边在 dfs 上端点之间的路径交删掉好边在 dfs 上端点之间的路径并。这里可以利用树上差分实现，具体看代码。

时间复杂度 $O(n+m)$
