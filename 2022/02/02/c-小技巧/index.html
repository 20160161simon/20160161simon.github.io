前言

主要记我遇到的一些 c++ 的特性和技巧。同时也会有点 oi 的技巧。

  1. c/c++ 中的逻辑运算符都是短路运算符

  2. while 是先判断再运算，do-while 是先运算再判断

  3. 测试时间方法

    当然，在 Linux 下也可以用 time 命令：
    e.g. time ./a 来测试运行 a 的时间

  4. scanf()可以返回成功输入的变量个数

  5. while(scanf() == 1)

    Windows 可以用 Enter -> Ctrl+Z -> Enter 停止
    Linux 可以用 Ctrl+D 停止

  6. 管道

    Windows 和 Linux 都提供了管道
    e.g. echo A.txt | B | C 表示从 A.txt 输入到可执行程序 B 再将 B 的输出丢到可执行程序 C 中运行

  7. 对拍
    Windows 版本

    Linux 版本

    Linux 脚本记得给个权限 chmod +x test.sh 这样才能运行 ./test.sh

  8. 一个以前错误的观点

    以前一直以为 #ifdef DEBUG 和 #endif 是配合开头的 #define DEBUG 的

    事实上是搭配编译选项 -DDEBUG 的

    PS：同理 #ifdef LOCAL 是搭配 -DLOCAL 的

  9. 鲁棒性（robustness）到底是啥？

    想起了之前在网上看到的一个段子：鲁棒性 = 耐操性。简直惊为天人
